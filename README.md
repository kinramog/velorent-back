Серверная часть Информационно-справочной системы проката велосипедов

Для запуска бэкенда, после скачивания проекта, необходимо выполнить следующие действия:
1. Выполнить команду `npm i` для установки пакетов и зависимостей.
2. Создать базу данных в PostgreSQL
3. Создать и заполнить файл `.env` в соответствии с примером в `.env.example`, указав данные созданной БД
4. Запустить проект с помощью команды `nest start`
